Function Reference

IgnitedRecord class

IR_RelProperty_has has_many('plural_name', [settings = array()])

Parameters:

type name desc.
string plural_name The name this relation will use
array settings The settings for the relation

Specifies a Has Many relation between this table and another.

IR_RelProperty_has has_one('singular_name', [settings = array()])

Parameters:

type name desc.
string singular_name The name this relation will use
array settings The settings for the relation

Specifies a Has One relation between this table and another.

IR_RelProperty belongs_to('singular_name', [settings = array()])

Parameters:

type name desc.
string singular_name The name this relation will use
array settings The settings for the relation

Specifies a Belongs To relation between this table and another.

object factory('table', [settings = array()])

Parameters:

type name desc.
string table The table the new instance will be mapped to
array settings The settings for the new instance

Creates a new IgnitedRecord instance.
Can be called statically.

object yamlfactory('filename')

Parameters:

type name desc.
string filename A filename or a yaml string

Creates a new IgnitedRecord instance from the settings stored in yaml data.
Can be called statically.

void add_hook('name', callable [, priority = 10])

Parameters:

type name desc.
string name A hook name
string
or array(&object, string) 		callable A callable which will be called when the hook with name executes
int priority The priority of the hooked function, lower number = higher priority

Adds a function / method to the hook name
